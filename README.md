# codex-sandbox

This repo contains a small Python 3 command-line utility, `frogdice.py`, for
rolling dice and printing the results.

## Requirements

- Python 3.8+

## Usage

Roll the default two six-sided dice:

```bash
python3 frogdice.py
```

Roll three eight-sided dice:

```bash
python3 frogdice.py 3 8
```

Sample output:

```text
Rolls: 2 5 (total 7)
```
